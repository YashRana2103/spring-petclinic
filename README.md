# spring-petclinic
The open source codebase [spring-projects/spring-petclinic](https://github.com/spring-projects/spring-petclinic) used for learning, understanding the architecture, design, and workflow. 

The step to master an ability to learning things from the technologies that are actually production based.

The [spring-projects/spring-petclinic](https://github.com/spring-projects/spring-petclinic) is a monolithic version of [spring-petclinic/spring-petclinic-microservices](https://github.com/spring-petclinic/spring-petclinic-microservices)
